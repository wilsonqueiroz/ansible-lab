# ANSIBLE


##### INSTALL AND CONFIGURE  ANSIBLE WITH VAGRANT.


### Installation

**CREATE DIRECTORY**
```sh
$ mkdir ansible && cd ansible
```
**CREATE VAGRANTFILE**
```sh
$ vagrant init
```
**PROVISIONING VIRTUAL MACHINE CENTOS/7**
```sh
$ vagrant up
```
**ACESSING VIRTUAL MACHINE CENTOS/7 WITH SSH**
```sh
$ vagrant ssh
```
